使用hexo快速搭建自己的博客

安装hexo

安装hexo之前，电脑上需要提前安装git和nodejs

建站

$ hexo init <folder> 初始化hexo项目
$ cd <folder>        切换到项目下
$ npm install        下载所需的依赖
新建完成后有以下目录
.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

配置

可以在 _config.yml 中修改大部份的配置。
一般会修改title，subtitle，author，description等配置，其它的一般不动
此时执行hexo generate和hexo serve，访问localhost：4000就可以看到搭建好的博客
hexo generate是生成静态文件，hexo serve是启动服务
注意：YAML依靠缩进来确定元素间的从属关系。因此，在_config.yml进行配置事请确保每个deployer的缩进长度相同，并且使用空格缩进。

要想使自己搭建的博客跟GitHub关联起来，并且用自己定义的地址来访问博客，需要做到以下操作

1.$ npm install hexo-deployer-git --save
2.在自己的GitHub上创建一个仓储，起一个.io的名字，
3.在 _config.yml中配置URL地址，改成http://+仓库的名字
4.在_config.yml文件的最下方配置deploy
    type: git
    repo: 仓库的地址
配置完成后执行hexo generate 和 hexo deploy //将网站部署到服务器
5.在浏览器上输入配置的URL地址就可以访问自己的博客了
