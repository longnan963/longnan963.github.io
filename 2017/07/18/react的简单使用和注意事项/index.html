react的简单使用和注意事项

react的安装使用

全局安装create-react-app

生成react项目

切换到项目名下

启动项目

在react中渲染页面

1.在js模块中首先引入react

import React from 'react'
import ReactDom from 'react-dom'
ReactDom 控制页面的内容，有两个参数：
第一个参数是渲染的dom元素，第二个参数是渲染的位置，在项目中的index.html中只有一个id为root的div，其他都是添加进去的虚拟dom元素

2.在js模块中会有render方法，且必须有返回值return

render方法中需要渲染的dom元素必须放在闭合标签中

3.组件

1.为了防止index.html中的代码了太大，不易查看，引入了组件
2.组件就是一个对象，先定义，再抛出，在需要引用的文件中导入就可以了
3.组件的名字首字母一般是大写，是为了跟dom元素区分开。
4.组件需要先继承react的方法和属性
5.在组件中定义变量的初始值，需要在state中定义，修改变量值用setState方法，
6.组件中的this必须指向组件名，方便调用，在事件中会改变this的指向，用bind(this)重新将this指向组件，在定时器中用箭头函数将this指向组件
7.组件中插入变量用{}操作

4.在项目中最重要的是参数的传递

1.父级向子级传递参数

1.先在子集组件中自定义属性，参数为属性值，
  <CommentList commentList={this.state.commentList}  />
2.在子集的组件中用props将传递的参数接收过来。但是需要先constructor和super调用一下父级react.Component的props方法
  constructor(props){
    super(props);
  }

2.子级向父级传递参数

子级向父级传递参数用自定义函数处理
1.在子集组件中自定义方法属性
    <CommentForm formHandler = {this.formHandler} />
2.在父级中声明定义方法formHandler
  formHandler(data){
     console.log(data);
  }
3.在子集中声明自己的方法并调用父级声明的方法formHandler传参数
   submitHandler(){
       var obj = {
           author:this.state.author,
           comment:this.state.comment
       }
       this.props.formHandler(obj);
   }

注意

1.在react中参数最好从根父级（app.js）向子集传递，通过props获取方便，如果从子集向父级传递参数，隔得层级比较多的时候很麻烦
2.处理ajax数据请求时在componentDidMount方法中，即组件加载完的状态，而不是放在render方法中，通过ajax请求改变列表commentList的值，再调用一次render方法重新渲染页面。
    $.ajax({
        url:'http://127.0.0.1:3100/api/comments',
        type:'get',
        success:(data)=>{
            this.setState({
                commentList:data
            });
        },
        error:function(err){
            console.log(err);
        }
    });
    在setState方法执行之后，会执行一次render方法
3.如果将ajax请求放在render方法中，那么只要修改render方法中的内容就会发送请求，不合理
