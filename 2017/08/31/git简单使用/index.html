git的简单使用

git的安装

Mac中git的安装

安装homebrew,然后借助homebrew

windows中git的安装

安装gitbash

查看git版本

git-version检查git是否安装成功

在git中设置自己的用户名和邮箱

git config --global user.name xiaolong
git config --global user.password zhaoxiaolong963@126.com

git config -l

查看设置的信息

初始化git仓库

git init
生成的.git文件千万不要删除

查看git版本仓库呈现的状态

git status

在当前文件夹下创建index.html,让git知道你添加了index.html

git add index.html 
git status 此时git已查找到你添加了index.html

提交代码

git commit -m'这是我第一次提交的代码'
每次在index.html中进行操作后，需要先add再commit

放弃这次修改

git checkout -- index.html

重置到之前提交的状态

git reset <commit状态码 3ac34> --hard

提交所有的操作

git add -A

创建分支

git branch test

切换分支

git checkout test

查看分支

git branch

删除分支

git branch -d test
